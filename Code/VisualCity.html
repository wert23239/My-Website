Visual City

A Visual C++ Game made in software design class.

This is a simple memory game. All you need to do is start the game and start placing tiles. As you place these tiles make sure you keep track of what you place where.

Need Image

Then you can press each the dice on top to start the game.

Dice Image

The key is to place all the tiles as fast as possible in the correct position and orientation. You know it is correct by listening for a sound.
